# EduQuiz Pro – Online Quiz & Certification System

## 📌 Overview

EduQuiz Pro is a responsive online quiz and certification system developed using HTML5, CSS3, and JavaScript. The application enables users to register, log in, take category-based quizzes, track their performance, and generate certificates upon successful completion. User data and quiz history are securely stored using the browser's Local Storage.

## 🚀 Features

* User Registration & Login
* Category-Based Quizzes
* Timer-Based Quiz System
* Question Navigation
* Progress Tracking
* Automatic Score Evaluation
* Quiz History
* Dynamic Certificate Generation
* Responsive Design
* Local Storage Data Management

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage
* Responsive Web Design

## 💻 How to Run

1. Clone this repository.
2. Open the project folder.
3. Run `index.html` in any modern web browser.
4. Register or log in to start the quiz.
